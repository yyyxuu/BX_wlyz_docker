# 用户心跳



![](https://bshare.optimix.cn/barCode?site=weixin&url=https%3A%2F%2Fwww.kancloud.cn%2Fli1446458045%2Fwlyz%2Fcontent%2F%25E6%258E%25A5%25E5%258F%25A3.md)



```
data = {"username":"用户名","password":"密码","token":"Token"}
Request：服务器地址  + "index.php/api/User/heart"
POST："id=" + 软件ID + "&data=" + 加密 (data)
```

#### 返回json**加密数据**

#### 解密后结果

```
{"code":"1006","data":""}
```

#### 结果说明

```
code:1006 心跳正常
code:1002 缺少参数
code:1001 用户名不存在
code:1003 密码错误
code:1004 Token不一致
code:1005 用户已经到期
code:1007 用户被强制下线
code:1008 Token已过期
```
