# 查询用户卡密



![](https://bshare.optimix.cn/barCode?site=weixin&url=https%3A%2F%2Fwww.kancloud.cn%2Fli1446458045%2Fwlyz%2Fcontent%2F%25E6%258E%25A5%25E5%258F%25A3.md)



```
data = {"card":"充值卡"}
Request：服务器地址  + "index.php/api/User/queryCard"
POST："id=" + 软件ID + "&data=" + 加密 (data)
```

#### 返回json**加密数据**

#### 解密后结果

```
{"code":"1303","data":{"name":"卡密名称","time":"10","time_type":"4","point":"100","create_time":"1497194831","comment":"卡密备注","state":"1","used":"2"}}
```

#### 结果说明

```
code:1303 查询成功
data.name 卡密名称
data.time 卡密时间 单位:小时
data.time_type  时间单位  1.秒 2.分 3.时 4.天 5.周 6.月 7.年
data.point 卡密点数
data.create_time 卡密生成时间戳
data.comment 卡密备注
data.state 卡密状态，1.正常使用 2.停止使用
data.used 卡密使用状态，1.未使用 2.已使用

code:1302 缺少参数
code:1301 充值卡不存在
```
